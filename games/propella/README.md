# PROPELLA

スマホを傾けて操縦する、コックピット視点のプロペラ機リングフライトです。90秒間にリングをつなぎ、スコアと残り時間を伸ばします。墜落はなく、島の山にぶつかると大きく弾かれて2秒間減速します。

## 起動

`index.html` を直接開いてもプレイできます。開発時はリポジトリの launch 設定 `propella`、または次のローカルサーバーを利用できます。

```sh
python -m http.server 8136 --directory games/propella
```

ブラウザで `http://localhost:8136/` を開きます。ジャイロはセキュアコンテキストが必要なため、実機では HTTPS の GitHub Pages 版を利用してください。

## ルール

- 制限時間は90秒です。
- 通常リングは `+100` 点・`+2秒`、15%で出現するゴールドリングは `+500` 点・`+5秒`です。
- リングを連続通過すると倍率が `×1.5 → ×2 → ×3 → ×4 → ×5` と上がります。次のリングを取り逃すとコンボが切れます。
- 気球に突っ込むと `+50` 点です。気球ではコンボは切れません。
- ブーストは満タンで3秒間。自然に毎秒8%、リング通過で30%回復します。
- ベストスコアはブラウザの `localStorage` に保存されます。

## 操作

### スマホ（ジャイロ）

1. スマホを縦向きにし、普段遊ぶ角度で持ちます。
2. タイトル画面の「ジャイロで飛ぶ」をタップします。
3. iOS 13以降では表示される「動作と方向へのアクセス」を許可します。
4. 3カウント後の端末姿勢がニュートラルとして記録されます。
5. 前後の傾きで上昇・下降、左右の傾きでバンク・旋回します。

傾き2°まではデッドゾーン、25°で最大入力です。プレイ中に右上の `◎ NEUTRAL` を押すと現在姿勢をニュートラルに再設定できます。画面の向きを変えた場合も再設定してください。

ジャイロが非対応または許可されなかった場合は、自動的にタッチ操作へフォールバックします。

### タッチ

- 画面をドラッグ: バーチャルスティック（上へドラッグで上昇、左右で旋回）
- 右下の `BOOST`: 押している間ブースト

### PC

- `W/S` または `↑/↓`: 上昇・下降
- `A/D` または `←/→`: 左右バンク・旋回
- マウス位置: 画面中心からの相対位置で操縦
- `Shift` または `Space`: ブースト
- `` ` ``: デバッグオーバーレイ表示

## デバッグ

`?seed=N` でコース乱数を固定し、`?debug=1` でFPS・内部状態を表示します。コンソールから次のAPIを利用できます。

### `window.__game`

- `getState()` — 位置、姿勢、速度、入力、スコア、コンボ、残り時間、リング・気球・山・雲、音声状態のスナップショット
- `dump()` — 現在のフライト状態をテキストで出力
- `step(ms)` — 自動更新停止中やrAF停止中でも指定時間を固定ステップで進めて描画
- `setInput({ pitch, roll, boost })` — `-1〜+1` の操縦入力を直接注入
- `clearInput()` — 注入入力を解除
- `teleport(x, y, z)` — 機体を移動
- `aimAtNextRing()` — 次のリングへ機首を向ける
- `passRing()` — 次のリングを強制通過
- `setTime(sec)` — 残り時間を変更
- `start()` — ready状態から開始
- `restart()` — スコアアタックを再開始
- `setAutoStep(enabled)` — rAF内のロジック自動更新を有効／無効化
- `forceBalloon()` — 検証用に生存中の気球へ移動し衝突処理
- `forceMountain()` — 検証用に山へ移動しバウンド処理
- `unlockAudio()` — AudioContextを初期化
- `testSfx()` — リング、ゴールド、気球、山、カウントダウン、雲、コンボのSFX経路を実行
- `getRendererConfig()` — シーンスケールから導出したcamera/fog設定を返す

### 描画ポンプ

- `window.__renderOnce(dt)` — 手動で1フレーム描画

例:

```js
__game.setAutoStep(false);
__game.start();
__game.setInput({ pitch: 0, roll: 0, boost: false });
__game.aimAtNextRing();
__game.step(3200);
__game.getState();
```
