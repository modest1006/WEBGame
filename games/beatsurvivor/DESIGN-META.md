# BEAT SURVIVOR メタ進行 設計書

対策1「メタ進行とアンロック」の設計。目標: **個性的で楽しい永続進行**。ユーザー方針: しっかりした設計で、ゲーム性を底上げする。

## コンセプト: 「サウンドシステムを組み上げる」

プレイヤーは場末のDJ。ランで稼いだ**ネオンチップ**で自分の**サウンドシステム（機材ラック）**を組み上げていく。強化が全部「音響機材」のフレーバーを持ち、世界観（ネオン×リズム）と完全に一致するのが個性。

- 進行の実感 = ラックに機材が増え、タイトル画面のDJブースが物理的に豪華になっていく（見た目に還元される永続進行）
- 「数値+5%」の羅列ではなく、**プレイスタイルを変える機材**を中心にする

## 通貨: ネオンチップ

- 獲得 = キル数×1 + GROOVEティア到達ボーナス(各50) + ボス撃破(300×周回) + 生存分ボーナス(10/30秒)
- 敗北してももちろん持ち帰れる（「次の1回」の動機）
- リザルトに「+◯◯チップ」の獲得演出（チップがジャラジャラ落ちるSE付き）

## 機材ラック（恒久強化）— 3カテゴリ×各3種

タイトル画面の「STUDIO」ボタンから機材ラック画面へ。各機材はLv3まで（コストは指数増）。

### 電源系（生存）
| 機材 | 効果 | フレーバー |
|---|---|---|
| 安定化電源 | 最大HP +10/20/30 | 「電源が細いと音が痩せる」 |
| 無停電電源 | 死亡時1回だけHP30で復活(ラン毎)/+雷無敵1s/+復活時ノヴァ | 「ライブは止められない」 |
| アイソレーター | 被ダメージ -5/10/15% | 「ノイズをぶった斬る」 |

### 出力系（攻撃）
| 機材 | 効果 | フレーバー |
|---|---|---|
| 真空管アンプ | 全ダメージ +6/12/18% | 「歪みこそ味」 |
| ツイーター | ビートショット弾速+20%/貫通+1/弾数+1 | 「高音は正義」 |
| バスレフ箱 | ノヴァ半径 +10/20/30% | 「箱鳴りで押し出す低域」 |

### 選曲系（スタイル・ここが個性の核）
| 機材 | 効果 | フレーバー |
|---|---|---|
| レコードバッグ | **開始武器を選択可能に**（Lv1:ビートショット固定解除→4武器から選ぶ/Lv2:開始Lv2/Lv3:開始パッシブ1つ付き） | 「今夜の1枚目」 |
| メトロノーム時計 | PERFECT判定 +8/16/24ms | 「体内時計を鍛える」 |
| ブースモニター | GROOVE減衰開始 +2/4/6ビート猶予 | 「自分の音が聴こえりゃ外さない」 |

計9機材・27段階。総コストはエンドレス10分級プレイヤーで15〜20ラン程度で完集する曲線（序盤は1〜2ランで1つ買える）。

## アンロック（実績連動）

チップとは別に、条件達成で自動解放される称号＋ご褒美:

- 「初ステージクリア」→ ENDLESSモード解放（※既存プレイヤーは解放済み扱い）
- 「GROOVE MAX到達」→ タイトルBGMにレイヤー追加
- 「エンドレス5:00」→ 機体色バリエーション（自機の色選択）
- 「エンドレス10:00 (NIGHT RIDER)」→ 称号表示＋レインボートレイル
- 「ボス3周回撃破」→ ボスラッシュ用の隠し色
- 実績はリザルトとSTUDIO画面で一覧表示（未達成はシルエット）

## 画面構成

- タイトル: PLAY(モード選択) / STUDIO / 実績。所持チップを常時表示
- STUDIO: DJブースの絵（購入済み機材が実際に描き足されていく）＋機材リスト。購入時は機材がラックにガシャンと収まる演出＋通電ランプ
- リザルト: 獲得チップ内訳 → 合計がカウントアップ → 「STUDIOで強化」導線

## セーブ

バッチ1のversion付きセーブ（beatsurvivor.save.v1）に `meta: { chips, gear: {...}, achievements: [...] }` を追加。マイグレーションは既存構造に自然追加（version 1のまま後方互換、フィールド欠落はデフォルト値）。

## バランス原則

- 機材はNORMALクリアを楽にしすぎない（合計でも初期比+30%程度の戦力）。エンドレスの到達時間を伸ばす方向に効く
- 「レコードバッグ」だけは戦力でなく**選択肢**を増やす機材で、これが最もリプレイを生む想定
- デバッグAPI: `giveChips(n)` / `buyGear(id)` / `unlockAll()` を追加

## 実装フェーズ

- Meta-1: 通貨獲得＋セーブ拡張＋STUDIO画面（機材9種）＋購入演出
- Meta-2: 機材効果のゲーム内適用＋実績システム＋タイトル/リザルト導線＋DJブース見た目進化
