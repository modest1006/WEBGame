# BUNNY DASH

夕焼けキャニオンを疾走するウサギの2Dプラットフォーマー。**スライディング着地→即ジャンプの「バニーホップ」で加速を重ねる**のが核メカニクス。タイムアタック制（ベストタイムは localStorage 保存）。

## 起動

**`index.html` をダブルクリックするだけで遊べます**（`file://` 直開き対応）。

ローカルサーバー経由でも可: `python -m http.server 8124 --directory games/bunnyhop` → http://localhost:8124/ （`.claude/launch.json` の `bunnyhop` 設定でも起動可）

## 操作

### キーボード（PC）
| キー | 操作 |
|---|---|
| ← → / A D | 移動 |
| Space / ↑ / W | ジャンプ（長押しで高く） |
| ↓ / S / Shift | スライディング |
| P / Esc | ポーズ |
| R | リスタート |
| M | ミュート |
| ` | デバッグオーバーレイ |

### タッチ（スマホ）
画面下ボタン: ◀ ▶ 移動 / スライド / ジャンプ

## メカニクス

- **スライディング**: 走行中に↓。開始時に一度だけ加速ブースト（クールダウン0.8秒）。低いトンネルはスライディングでしか通れない
- **バニーホップ**: スライディング状態で着地して0.1秒以内にジャンプすると速度を維持したまま+55加速（コンボ加算）。着地後0.15秒地上にいるとコンボ消滅・超過速度も減衰するので、**跳び続ける限り速い**
- 通常走行の最高速は280、バニーホップで最大680まで伸びる。中盤の8タイル大ジャンプは加速しないと越えられない（スライドブースト+ホップ1回が最低ライン)
- コヨーテタイム0.1秒・ジャンプ先行入力0.13秒・可変ジャンプ高対応

## URLパラメータ

- `?debug=1` — デバッグオーバーレイ表示（座標・速度・接地・コンボ等）

## デバッグAPI（`window.__game`）

セルフレビュー・自動検証用。決定論的にテストする場合は `pause()` してから `hold()`/`step(ms)` で進める。

| API | 説明 |
|---|---|
| `getState()` | 位置・速度・状態のスナップショット |
| `dump(radius?)` | プレイヤー周辺のASCIIマップ（R=自機, X=地形, ^=トゲ, o=🥕, C=CP, G=ゴール） |
| `step(ms)` | 時間をms分進める（ポーズ中でも可） |
| `hold({left,right,slide})` | 入力の押しっぱなし状態を注入 |
| `pressJump() / releaseJump()` | ジャンプ入力 |
| `teleport(tx, ty)` | タイル座標へ瞬間移動 |
| `setVel(vx, vy)` | 速度を直接設定 |
| `respawnAt(tx, ty)` | リスポーン地点変更 |
| `die()` | 即死（リスポーン検証用) |
| `start() / pause() / resume()` | 状態遷移 |

検証例（バニーホップでの加速確認）:

```js
__game.start(); __game.pause();
__game.hold({ right: true });
__game.step(1500);                 // 助走（maxRun=280に到達）
__game.hold({ right: true, slide: true }); // スライディング（+ブースト）
__game.step(50);
__game.pressJump(); __game.step(600);      // ホップ1
__game.pressJump(); __game.step(600);      // 着地際に先行入力→ホップ2
__game.getState();                 // vx が 280 を超えて伸びていればOK
```
