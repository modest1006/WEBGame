# 星喰い HOSHIKUI

「コロガリ魂」の宇宙スピンオフです。岩石から始まる天体を操作し、小天体を喰って質量を増やし、最後はブラックホールとして銀河コアを飲み込みます。

## 起動

- 直接開く: `games/hoshikui/index.html`
- ローカルサーバー: `python -m http.server 8132 --directory games/hoshikui`

## 操作

- WASD / 矢印キー: カメラ基準で移動
- マウス右側ドラッグ / 画面右側ドラッグ: カメラ回転
- 画面左側ドラッグ: モバイル用スティック
- R: リスタート
- M: ミュート
- P / Escape: ポーズ
- バッククォート: デバッグ表示

## ルール

自分の質量が相手の 0.9 倍を超えている天体は、接触すると捕獲できます。捕獲した天体は表面に貼り付かず、質量順の層に分かれてプレイヤーの周囲を衛星として公転します。表示衛星は最大 40 個で、超過分は小さいものから質量へ統合されます。

進化段階は `岩石 -> 小惑星 -> 惑星 -> 巨大惑星 -> 恒星 -> 超巨星 -> ブラックホール` です。進化ごとに制限時間が延長され、最終段階で銀河コアを喰うとクリアです。

## デバッグ API

`window.__game`:

- `getState()`
- `dump()`
- `step(ms)`
- `move(x, z)`
- `teleport(x, z)`
- `setMass(m)`
- `absorbNearest(n)`
- `evolve()`
- `win()`
- `finish()`
- `validate()`

URL クエリ:

- `?debug=1`: デバッグオーバーレイ
- `?seed=N`: 配置シード指定

`window.__renderOnce(dt)` は 1 フレーム描画して canvas の DataURL 長を返します。
